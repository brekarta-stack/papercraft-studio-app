# 페이퍼크래프트 스튜디오 (Papercraft Studio)

**내장 디자인·사진·3D 모델을 자르고 접는 종이공예 전개도(페이퍼크래프트)로 바꿔 주는 무료 데스크톱 프로그램.**

👉 **소개 · 사용법 · 다운로드 : https://papercraft.kr/download**

## 받기 (Windows)
1. [최신 릴리스](../../releases/latest)에서 `PapercraftStudio-windows-x64.zip` 를 받습니다.
2. 압축을 풀고 `PapercraftStudio.exe` 더블클릭 — Windows 10/11 (64-bit), Python 설치 불필요.
3. 처음 실행 시 ‘Windows의 PC 보호’ 파란 창이 뜨면 **‘추가 정보 → 실행’**.

## 이런 걸 할 수 있어요
- **140종 내장 디자인** — 동물·공룡·탈것·세계/한국 건축물·오브젝트
- **사진 → 종이 스탠드**, **내 3D 파일(OBJ·STL·PLY·GLB) → 전개도**, **AI 텍스트 → 3D 생성**
- 자르기/산접기/골접기 구분 · 접착 번호 짝 · 풀칠 방향 화살표 · **확대해도 안 깨지는 벡터 도면**
- **실측 1:1 인쇄 PDF**(A0~B6, 조립 지도·스케일바) · **커터용 SVG/DXF**(칼선/오시선 레이어, 도무송 모드)
- 그림 사용설명서(PDF) 내장 — 초등학생도 따라 만들 수 있어요
